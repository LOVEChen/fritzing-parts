# Arduino Pro mini
通常のArduino Pro miniの部品定義を加工した
独自の部品定義．

## 部品の形状
通常のPro miniの部品定義では，プログラムの書き込みなどに
利用するシリアルの端子も定義されており，ブレッドボード
ビューやプリント基板の配線時にこれらの端子が邪魔になる．
(これらの端子は浮かせて，USBシリアル回路をつなげて
使うことが多く，プリント基板に落としたくない．)
